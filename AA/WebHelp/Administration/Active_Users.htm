Active Users page

The Active Users page lists the users whose current status is Active. It also displays registration data for each user. This data includes:

  • Name

  • Country and city where currently assigned

  • Phone number

  • Email

  • Role, with regard to the Alumni Archive. List of roles

  • Any additional countries in which they are active

  • Current status (Active, Newly Registered, Inactive, etc.)

  • Office Code

  • Comments

The colored option buttons on the right can be used to change a user's status, by clicking on the Deny or On Hold buttons.

Clicking on the person's name will open the User Profile page, displaying user registration data for that person.
