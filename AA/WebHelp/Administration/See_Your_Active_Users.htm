See Your Active Users page

The See Your Active Users page lists all of the users from your area whose current status is Active. It also displays registration data for each user. This data includes:

  • Name

  • Country and city where currently assigned

  • Phone number

  • Email

  • Role, with regard to the Alumni Archive. List of roles

  • Any additional countries in which they are active

  • Status (Active)

  • Office Code

  • Comments

Clicking on a user's name will open the User Profile page, displaying the registration data for that person.

The current page can also be used if, for example, you need to remove or suspend access for a user. Click on the colored option buttons on the right can be used to change a user's status.

  • When a user's status is changed to On Hold, they cannot log on to the Archive. Their name will appear on your list on the Approve Changes page, where their status can be changed by using the Deny or Approve buttons there.

  • Clicking on the Deny button will drop the user from the Archive users list, and they will no longer have access the Archive. To regain access, they would have to re-register and be approved.
