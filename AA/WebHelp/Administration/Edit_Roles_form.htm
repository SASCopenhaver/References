Edit Roles -- form page

The Edit Roles form page is used to edit existing roles or create new roles. It is opened from the Edit Roles list page by clicking on the Add New Role button (to add a role) or one of the Edit controls (to edit the selected role).

  • See Edit Roles for more information.

To add a role

  1. The Role Name list box (on the left) will display Create New Role. In the empty text box to the right, type in a name for the new role.

  2. Type in a Role Description.

  3. Select an Authorizing Agent Role from the drop-down list.

  4. Under Action Page Access and Utility Page Access are pairs of Yes or No option buttons, for indicating whether this role is to have administrative access to each of the listed pages. By default these will all be set to No. Click on the Yes buttons for the pages that the role should be able to access.

  5. When finished, click on Submit, at the lower right.

To edit a role

  1. This window will open with the current data for the role filled in.

  2. If you select a different Role Name, the other data (including the option buttons) will change to reflect that role.

  3. If you want to re-name the role, type that name in the text box on the right.

  4. You can edit the Role Description.

  5. You can select a different Authorizing Agent Role from the drop-down list.

  6. You can select different option buttons, under Action Page Access and Utility Page Access. These control whether the role will have administrative access to each of the listed pages

  7. When finished, click on Submit, at the lower right.
