Edit Roles

Two pages are used by administrators to add or change Alumni Archive user roles. The Edit Roles list page is the first page.

It displays a table that lists the current user roles. This includes, for each role:

  • The role name.

  • The role description.

  • The required user role for the authorizing agent -- the person who can confirm users who register for the listed role.

  • The fourth column, Actions, has two controls for each role.

    • Edit opens the Edit Roles form page. See below for more information about editing roles.

    • Delete will delete the role from the database, after confirmation.

To add a role

  1. Click on the Add New Role button at the lower right of the list page.

  2. When the small dialog window opens, click on OK.

  3. This opens the Edit Roles form page.

  4. The Role Name list box (on the left) will display Create New Role. In the empty text box to the right, type in a name for the new role.

  5. Type in a Role Description.

  6. Select an Authorizing Agent Role from the drop-down list.

  7. Under Action Page Access and Utility Page Access are pairs of Yes or No option buttons, for indicating whether this role is to have administrative access to each of the listed pages. By default these will all be set to No. Click on the Yes buttons for those pages that the role should be able to access.

  8. When finished, click on Submit, at the lower right.

To edit a role

  1. Identify the role to be edited.

  2. In the far right Actions column for the role, click on Edit.

  3. This opens the Edit Roles form page, with the current data for the role filled in.

  4. You can select a different Role Name; if you do, the other data (including the option buttons) will change to reflect that role.

  5. If you want to re-name the role, type that new name in the text box on the right.

  6. You can edit the Role Description.

  7. You can select a different Authorizing Agent Role from the drop-down list.

  8. You can select different option buttons, under Action Page Access and Utility Page Access. These control whether the role will have administrative access to each of the listed pages

  9. When finished, click on Submit, at the lower right.
