Quick Search

The Quick Search page allows users to search for names and other data of other Alumni Archive users. The search criteria must include one or more of the following: Country, Role, and Last Name.

Initially, the page displays only the three checkboxes (Country, Role, and Last Name), and two buttons (Search and Reset).

  1. Mark the checkboxes for the criteria you wish to use. A list box or text box will be displayed to the right of each marked checkbox.

  2. For Country or Role, click on the List button at the right of the list box, to drop down the list of countries or roles.

    1. Or press a letter key to drop down the list and move the focus (highlighting) to the first list item that begins with that letter.

  3. If necessary, scroll to find the desired country or role, then click on it to select it.

  4. For Last Name, click or tab into the text box, and type in the last name.

    •  Be careful with spelling. Double check it if the search fails to return the expected results.

  5. After entering criteria, click on the Search button at the bottom.

  6. When the Submit Request box appears, click on OK to run the search.
