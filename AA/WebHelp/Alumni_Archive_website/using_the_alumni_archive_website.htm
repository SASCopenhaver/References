Using the Alumni Archive Website

This website provides access to the Alumni Archive database, which houses data on past participants from all of the ECA funded exchange programs. Specific alumni can be located and their contact data can be updated. Website users can also use the website to generate contact lists, mailing labels, statistical reports, and ad hoc reports.

This website needs to have current contact data and accurate bio data for all alumni. You are strongly encouraged to update the alumni data here whenever you have newer or better information.  

Users must first register and be granted access. Each user is assigned a role for their Alumni Archive use, based on their position and current duty station. See Registration for more details.

Users are encouraged to check and insure that the data stored here for all of their alumni is as current and accurate as possible. See Updating Alumni Data for details of this process.

Searches are available for locating records for updating, and for generating reports, contact lists, mailing labels, etc. The process is similar in all cases. You can save your search criteria as named Saved Searches, for re-use at a later date.

  • Update searches are used when updating alumni records. Because the downloading and display of large numbers of records can take a long time, records will be displayed on the results page only if no more than 500 records are found matching your search criteria.

  • Create and Print searches are used when generating reports, contact lists, and mailing labels. These allow you to search on a few more fields, and are not limited in size.

The Alumni Archive can be used to generate a number of defined reports, or users can design their own custom reports. Contact lists and mailing labels can be easily created.

For users with disabilities, see the basic tips listed in Accessibility.

If you find any bugs or have any other problems with the Archive or the website, or if you have any suggestions or comments, please send them to the Archive Administrator, at AlumniArchive@state.gov.
