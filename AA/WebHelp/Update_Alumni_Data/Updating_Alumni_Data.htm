Updating Alumni Data

To update the archived data for one or more alumni:

  1. Open the Update Alumni Data page, to search for the records for those alumni.

  2. Enter a name and / or other search criteria. Or select a saved custom search.

    1. See Update Alumni Data for more details on specifying criteria values.

  3. Click on the Search button at the upper right of the page.

  4. If no records are found, or if more than 500 records are found that match the search criteria, an Information page opens displaying the number of records found, and inviting you to re-define your search criteria. Click on the OK button to return to the Update Alumni Data search page.

  5. If the search finds at least 1 record that matches the criteria, and not more than 500, the Search Results page is opened.

    1. If  1 - 25 records are found, this page displays all of the results, listed alphabetically.

    2. If 26 - 500 records are found, the page displays the first 25, along with links for displaying the others, in sets of 25.

  6. While the default order for the list is alphabetical by last name and first name, you can re-sort the list by clicking on any of the column headers.

    1. Clicking on any of the column headers will re-sort the list, using the values in that column. Clicking a column header again toggles the sort order between ascending order (e.g., oldest year first) and descending order.

  7. Locate the name of an alumna(us) you want to update.

  8. Click on that name to open the Alumni Profile page, displaying the data for the selected person.

    1. Some program data is displayed at the top of the page, read-only.

    2. The main panel displays the currently available participant data, in 38 editable controls (text boxes, list boxes, and checkboxes).

  9. Correct the existing data, or add any missing data that you have.

  10. After updating the record, click on the Save button at the bottom of the page to save your changes to the database.

  11. To save data and return to the Search Results page, click on the Save & Close button.

  12. To return to the Update Alumni Data page, to create a new search, click on the Update Alumni Info link, on the global navigation bar at the top of the page.
