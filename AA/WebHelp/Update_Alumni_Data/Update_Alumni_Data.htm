Update Alumni Data page

The Update Alumni Data page is used to find the alumni records you want to update. You enter values for one or more of the search criteria, then click on the Search button to run the search. Or, if you have saved the search criteria from any of your previous searches, you can choose one of those saved searches.

If the search finds from 1 to 500 records, these alumni are then listed on the Search Results page. From there you can click on a name to open an Alumni Profile page, displaying data for that alumnus(a). You can easily return to the results page to select other alumni listed there and update their data, or return to the Update Alumni Data page to run another search.

Define the search

The page has three sections for defining the search.

  1. The Basic section offers the seven basic search criteria, and is always displayed.

  2. The Additional Criteria section is located below the main section, and by default is hidden. There is a control link at the bottom of the Basic section for displaying or hiding this page section. This section offers six additional criteria that can be used to define a search.  

  3. The Saved Searches section lists your saved searches. When it is displayed, it is located at the top of the page, above the Basic section; but by default it is hidden. There is a control link above the Basic section for displaying or hiding this list. These saved searches are sets of search criteria can be re-used, either as is or with modifications.

Basic search section.

This is the central section of the page; it is always displayed. Most searches can usually be accomplished using one or more of the seven criteria displayed here:

  • First Name and Last Name

    1. When you enter a first or last name as a search criterion, the search looks for all alumni whose names begin with that string of letters.

    2. For example, searches for alumni named Smith will also include any alumni named Smithson in the results, if they also fit the other criteria.  And searches for Bert will also return anyone listed as Bertram.

  • Gender

  • Country From

    1. For American participants being sent abroad, this is the United States. For exchange participants coming to the U.S., this could be either the country of residence, the citizenship country, or the funding country, depending on the data that was collected and provided by the program on which they participated (or on subsequent updates).

      You can select one or more countries.  

  • Country To

    1. This is the country to which they traveled as a program participant. Some alumni may have traveled to more than one country on ECA programs.

      You can select one or more countries.

  • Program Type

  • This is the ECA exchange program on which a participant traveled.

    • Note that the names of some programs have changed over the years. If you select a program that has had two or more names, the search will automatically include the other program names: for example, the International Visitor Program and the International Visitor Leadership Program.

    • You can select more than one program.

      • To select several programs, click on one to highlight it, then hold down the <Ctrl> key while clicking on the others.

      • To select several programs that are listed sequentially, such as all Fulbright programs, highlight the first (or last) program in that sequence, then hold down the <Shift> key while selecting the last (or first) program type in that sequence.

  • Fiscal Year or Program Start Year

  • You can limit the search to one or more specified years.

    • Select either the fiscal (funding) year of the program, or the calendar year in which it started. 

    • Some alumni have participated on multi-year programs, such as the Fulbright Student Program, and each year may be listed separately in the search results.

    • After selecting a type of year, you must select one or more years in the list box on the right.

      • To select several individual years, click on one year and then hold down the <Ctrl> key while clicking on the other years.

      • To select a span of years, click on the first (or last year) in the span, then hold down the <Shift> key while clicking on the last (or first) year.

      • You can combine these two methods.
         

To select more than one Country, Program, or year to search on:

    • Click on one to highlight it, then hold down the <Ctrl> key while clicking on the others.

    • To select several that are listed sequentially, highlight the first (or last) in that sequence, and then hold down the <Shift> key while selecting the last (or first) in that sequence.

    • You can also combine these two actions: select a sequence and then use the <Ctrl> key to add items from elsewhere on the drop-down list.
       

Additional criteria section

To display the lower section of the page, with additional search criteria, click on the Show Additional Criteria link located at the bottom of the Basic section.

  • That link is then replaced with the Hide Additional Criteria link, which can be used to hide this page section, if not needed.

This page section has six other criteria that can be included in your search: Field of Study/Project Subject, Citizenship, Country of Residence, City of Residence, Host Institution, and City Visited.

  • Citizenship and Country of Residence are list boxes with drop-down lists to select from. The other four criteria use text boxes.

  • You can only specify a value for one of the four text string criteria (Field of Study/Project Subject, City of Residence, Host Institution, and City Visited) for any one search. Trying to use more than one of these criteria will cause the search to fail. This limit is imposed because of the time and web server resources that are required for this type of search.
     

Saved searches section

Saved searches are saved sets of values for selected criteria, which can be re-used at a later time. They are listed in the upper section of the page.

By default this page section is hidden; to display it, click on the Show Saved Searches link at the upper left of the page.

  • That link is then replaced by the Hide Saved Searches link, which can be used to hide this page section, if not needed.

To use a saved search

  1. Display the list.

  2. Click on one of the listed searches.

  3. The values for each of the included criteria will be loaded into the appropriate controls (text or list boxes) in the search form.

  4. If the search uses any of the Additional Criteria (see above), that lower page section will be displayed.

  5. You can alter any of these values, and you can add values for any criteria that were not included in the saved search.

  6. Click on the Search button to run the search.

To create a new saved search

  1. Enter values for the criteria you want to use, and then click on the Search button.

  2. A pop-up dialog window will open, asking if you want to save these as a saved search.

  3. Click on Yes.

  4. A second pop-up dialog will ask you to enter a name for the custom search.

    • If you selected an existing saved search and then modified it, the name of the original saved search will be displayed here. Leave it if you want to replace the original search, or enter a new name if you want to keep both versions of the search.

  5. Enter a name and click OK.

  6. The search's criteria values will be saved under that custom search name, and the search will start.

 

To run a search

  1. Enter values for the selected criteria. This can be a saved search, a modified saved search, or a new search.

  2. Then click on the Search button.

  3. If you are running a previously saved search without adding or changing any criteria values, the search will run immediately.

  4. In all other cases, a pop-up dialog window will open, asking if you want to save this search.

    1. Click either Yes or Cancel.

    2. If you choose Yes, a second pop-up window asks you to enter a name for the custom search. Enter a name and click OK. The search's criteria values will be saved under that name, and the search will run.

    3. If you click Cancel, the search will run immediately.

  5. After the search runs:

    • If the number of records found matching the criteria is at least one and not greater than 500, the Search Results page will open, displaying the list of alumni.

    • If the number is zero or is greater than 500, the Information page opens, telling you how many records were found, and offering a link back to the Update Alumni Data page, to re-define the search.

  6. Click on a alumni name on the Search Results page to open the Alumni Profile page, which has the form used for updating.
