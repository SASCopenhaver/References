Alumni Profile page

The Alumni Profile page is used to update alumni information. It is opened by clicking on an alumni name on the Search Results page -- it displays the data for the selected alumna(us).

Some program data is displayed at the top of the page, read-only.

The main panel displays the currently available participant data, in 39 editable controls (text boxes, list boxes, and checkboxes).

  1. Please correct the existing data, and/or add any missing data that you have.

  2. Click on the Save button at the bottom of the page to save your changes in the database.

  3. To save data and return to the Search Results page, click on the Save & Close button.

  4. To return to the Update Alumni Data page, use the global navigation bar at the top of the page.
