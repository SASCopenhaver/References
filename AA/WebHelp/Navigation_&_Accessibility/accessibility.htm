Accessibility

Because you are using Windows 2000 or Windows XP, and are viewing the Alumni Archives webpages in an Internet Explorer browser window, the accessibility options that are built into Windows and Internet Explorer are available to you. These features may make it possible, or easier, to use the Archive if you have impaired hearing, vision, or movement. Some users may need the use of additional hardware or software.

Windows XP

To read about Windows XP accessibility options, open Windows Help.

  1. Open Windows Explorer by clicking on the My Computer icon on your desktop.

  2. On the menu bar at the top, click on Help and select Help and Support Center from the drop-down list.

  3. When the Help and Support Center window opens, in the left column (Pick a Help Topic) click on Accessibility. to display the list of accessibility topics.

  4. Click on a topic to display it.

Windows 2000

  1. Open Windows Explorer by clicking on the My Computer icon on your desktop.

  2. On the menu bar at the top, click on Help and select Help Topics from the drop-down list.

  3. When the Help window opens, the Contents tab should be the default in the left panel. If not, click on it.

  4. In the Contents tab, click on Accessibility for Special Needs to open that book/folder.  

  5. Click on any of the topics to display it.

Internet Explorer

  1. On the Internet Explorer menu bar at the top of this window, click on the Help menu and select Contents and Index from the drop-down list.

  2. When the Help window opens, the Contents tab should be the default tab in the left panel. If not, click on it.

  3. In the Contents tab, click on Accessibility to display the available topics.

  4. Click on a topic to display it.
