Update Alumni Data page

The Update Alumni Info page is used to find the alumni records you want to update. You enter values for one or more of the search fields, then click on the Search button to run the search. Or, if you have saved the search criteria from any of your previous searches, you can choose one of those saved searches.

  • Note that the saved searches preserve only the search criteria, not the search results. Running a saved search again, at a later date, may produce different results because the records in the Alumni Archive may have changed.

If your search finds from 1 to 500 records, the those alumni are then listed on the Search Results page. From there you can click on a name to open an Alumni Profile page, displaying the data for that alumnus. You can easily return to the results page to select other alumni listed there and update their data, and you can return to the Update Alumni Info page to run another search.

  • If you want to create a list of more than 500 alumni, go to the Create Contact List page or the Create Mailing Labels page, to create a list in an Excel spreadsheet. The number of alumni that can be included on those lists is not limited.

Define the search

There are two ways to enter the search definition; they can be combined.

  1. You can specify the search criteria, using the search form in the main section of the page. It offers ten fields that you can search on.

  2. The Saved Searches section lists your saved searches. These are sets of search criteria that can be re-used, either exactly as saved or with modifications. When you click on one of these, it loads the saved search criteria into the appropriate field controls in the search form. See Saved searches, below.

    Note that the Saved Searches section of the page is usually hidden, by default. There is a Show Saved Searches link above the main section for displaying or hiding the Saved Search list. It displays above the main section.

Search criteria

The search form is the main section of the page; it displays controls for ten fields. You must enter or select a value for any one or more of these fields. What this does is to limit the search results to only those alumni records that have the specified value or values for each of the selected fields. All non-selected fields are ignored; they can have any value.

The available fields are:

First Name and Last Name

  • When you enter a first or last name as a search criterion, the search looks for all alumni whose names begin with that string of letters.

    • For example, searches for alumni named Smith will also include any alumni named Smithson in the results, if they also fit the other criteria. And searches for Bert will also return anyone listed as Bertram.

Gender - male or female.

Country From

For American participants being sent abroad, this is the United States. For exchange participants from abroad, this is their country of residence.

  • You can select more than one country. To select several countries, click on one to highlight it, then hold down the <Ctrl> key while clicking on the others.

Country To

This is the country to which they traveled as a program participant. Some alumni may have traveled to more than one country on ECA programs.

  • You can select more than one country. To select several countries, click on one to highlight it, then hold down the <Ctrl> key while clicking on the others.

Program Type

This is the ECA exchange program on which an individual participated.

  • Note that the names of some programs have changed over the years. If you select a program that has had two or more names, the search will automatically include the other program names: for example, the International Visitor Program and the International Visitor Leadership Program.

  • You can select more than one program.

    • To select several programs, click on one to highlight it, then hold down the <Ctrl> key while clicking on the others.

    • To select several programs that are listed sequentially, such as all Fulbright programs, highlight the first (or last) program in that sequence, then hold down the <Shift> key while selecting the last (or first) program type in that sequence.

    • You can also combine these two actions: select a sequence and then use the <Ctrl> key to add items from elsewhere on the drop-down list.

Fiscal Year or Program Start Year

This list box allows you to select which of these two fields you want to search on. You must limit the search to one or more specified years.

  • Select either the fiscal (funding) year of the program, or the calendar year in which it started. 

  • Some alumni have participated on multi-year programs, such as the Fulbright Student Program, and each year may be listed separately in the search results.

  • After selecting a type of year, you must select one or more years in the list box on the right.

    • To select several individual years, click on one year and then hold down the <Ctrl> key while clicking on the other years.

    • To select a span of years, click on the first (or last year) in the span, then hold down the <Shift> key while clicking on the last (or first) year.

    • You can combine these two methods.

Citizenship and Country of Residence

These are list boxes with drop-down lists of countries.

  • You can select more than one country. To select several countries, click on one to highlight it, then hold down the <Ctrl> key while clicking on the others.

  

Saved searches

Saved searches are saved sets of values for selected fields, which can be re-used at a later time -- they do not include any saved results. Saved searches are listed in the upper section of the page.

  • By default this page section is hidden; to display it, click on the Show Saved Searches link at the upper left of the page.

    That link is then replaced by the Hide Saved Searches link, which can be used to hide this page section, if not needed.

To use a saved search

  1. Display the list.

  2. Click on one of the listed searches.

  3. The values for each of the included criteria will be loaded into the appropriate fields (text or list box controls) in the search form.

  4. You can alter any of these values, and you can add values for any criteria that were not included in the saved search.

  5. Click on the Search button to run the search.

To create a new saved search

  1. Enter values for the criteria you want to use, and then click on the Search button.

  2. A pop-up dialog window will open, asking if you want to save these as a saved search.

  3. Click on the OK button.

  4. A second pop-up dialog will ask you to enter a name for the search.

    • If you selected an existing saved search and then modified it, the name of the original saved search will be displayed here. Leave it if you want to replace the original search, or enter a new name if you want to keep both versions of the search.

  5. Enter a name and click Save and Search.

  6. The search criteria will be saved under that search name, and the search will start.

To delete a saved search

  1. Click on the Delete button at the right of the saved search. A confirmation dialog opens.

  2. Click on OK to complete the deletion.

  3. An information message opens, confirming the deletion. Click on OK to return to the Update Alumni Info page.

Run a search

  1. Enter values for the selected criteria. This can be a saved search, a modified saved search, or a new search.

  2. Then click on the Search button.

  3. If you are running a previously saved search without adding or changing any criteria values, the search will run immediately.

  4. In all other cases, a pop-up dialog window will open, asking if you want to save this search. Click either OK or Cancel.

    1. If you choose OK, a second pop-up window asks you to enter a name for the search. Enter a name and click on Save and Search. The search's criteria values will be saved under that name, and the search will run.

    2. If you click Cancel, the search will run immediately.

  5. After the search runs:

    • If the number of records found matching the criteria is at least one and not greater than 500, the Search Results page will open, displaying the list of alumni.

    • If the number is zero or is greater than 500, the Information page opens, telling you how many records were found, and offering a link back to the Update Alumni Info page, to re-define the search.

  6. Click on a alumni name on the Search Results page to open the Alumni Profile page, which has the form used for updating.
