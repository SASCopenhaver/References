Search Results page

The Search Results page displays a list of the alumni whose records were found. Also, at the top of this page are the search criteria that were used to produce this list.

  • This page is displayed only if from 1 to 500 alumni records were found. Otherwise, an Information page tells you how many records were found and offers to return you to the search page, to re-define the search.

  • This page can only display 25 alumni at a time. Therefore, if the search finds more than 25 alumni (but no more than 500), the first 25 are displayed initially, along with links for displaying each of the other sets of 25.

For each of the alumni, the results table lists their Full Name, Field of Study/Project Subject, Program Type, Program Fiscal Year, Program Start Year, and whether their record has been updated since it was first entered. If it has been updated, the date of the last update is listed.

  • The initial order for the list is alphabetical, by name (last name, then first name). Clicking on any of the column headers will re-sort the list, using the values in that column. Clicking a column header again toggles the sort order between ascending order (e.g., oldest year first) and descending order.

Click on a name to open the Alumni Profile page, with the data for the selected alumna. That page is where you can update the alumni data.

To return to the Update Alumni Data page, click on the Back To Search Criteria button, above the alumni list.
