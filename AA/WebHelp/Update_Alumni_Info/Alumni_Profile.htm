Alumni Profile page

The Alumni Profile page is used to update alumni information. It is opened by clicking on an alumnus name on the Search Results page -- it displays the data for the selected alumna(us).

Some program data is displayed at the top of the page, read-only.

  • If you believe that any of the program data is incorrect, please send your suggested change to the Alumni Archive data administrator, at AlumniArchive@state.gov. The administrator will seek the approval for this change from the appropriate ECA Program Office and, once it is approved, will make the change in the Archive database.

The main panel on this page displays the currently available participant data, in 40 editable field controls (text boxes, list boxes, and checkboxes).

To update an alumnus:

  1. Please update the existing data. Make corrections, and add any missing data that you have.

  2. Please note that, because of the original form of much of the legacy data that has been preserved in the Alumni Archive, the whole of the Residence Address and Business Address for the older Alumni data have been stored in their respective Street Address fields.

    In some cases, the quality of the old data has resulted in this field being badly formatted, and unusable for mailing labels, and reports. If this is true for any of your Alumni, you will need to correct the form of the data (for example, by adding carriage returns), in addition to making any needed updates, before you can create mailing labels, or any contact lists or reports that use address data.

    Separate fields have been provided for the Institution (for Business addresses only), for the City, State or Province, and Country, and for the Postal Code (if any). These are being used for all newer alumni data. So when you correct or update older Alumni data, please use these fields and put only the current street address data in the Street Address fields.

  3. For the ELP (English Language Proficiency) ratings, under Additional Information at the bottom of the page, you can use the following scale:

    1. 0 = No Proficiency

    2. 1 = Elementary Proficiency.

    3. 2 = Limited Working Proficiency

    4. 3 = General Professional Proficiency

    5. 4 = Advanced Professional Proficiency

    6. 5 = Native or Bilingual Proficiency

  4. After updating the record, click on the Save button at the bottom of the page to save your changes to the database.

  5. To save your data and return to the Search Results page, click on the Save & Close button.

  6. To return to the Search Results page without saving, click on the Close Without Saving button.

  7. To return to the Update Alumni Data page, to create a new search, click on the Update Alumni Info link, on the global navigation bar at the top of the page.
